# GNN IPinfo

A premium, minimalist WordPress plugin to display visitor IP information using the IPinfo.io API. Built with performance, security, and universal theme compatibility in mind.

## Features
- **Premium UI:** Modern Glassmorphism design that adapts to any Light or Dark theme.
- **Minimalist Layout:** Compact single-column display for seamless theme integration.
- **Copy to Clipboard:** One-click IP copying for user convenience.
- **Performance:** 1-hour transient caching to minimize API calls and boost speed.
- **Security:** Nonce-protected manual updates and strict output escaping.
- **GitHub Updates:** Built-in updater for easy management via GitHub Releases.
- **Debug Mode:** Optional raw API data view for administrators.

## Installation
1. Upload the `gnn-ipinfo` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to `Settings > GNN IPinfo` and enter your [IPinfo.io](https://ipinfo.io) API token.

## Usage
Simply use the shortcode `[gnn_ipinfo]` anywhere in your posts, pages, or widgets.
